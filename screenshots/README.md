## How to capture in Power BI Desktop

1. Open `EnterpriseFinancialPortfolio.pbix`
2. Navigate to each page
3. Press `Windows + Shift + S` to capture a screen region
4. Or use `File → Export → Export to PDF` for all pages at once
5. Save each screenshot to this folder with the naming convention above

## Optimal settings for GitHub README display

- Resolution: 1920 × 1080 or 2560 × 1440
- Format: PNG (preferred) or JPEG
- DPI: 96 (screen resolution)
- Power BI Desktop zoom: 100%
